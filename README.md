# 🤖 AI Resume Analyzer

An AI-powered web app that analyzes resumes and suggests suitable job roles.

## 🚀 Features
- Resume skill extraction
- Job role matching
- Simple UI using Streamlit

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas / NumPy

## ▶️ Run Locally
pip install -r requirements.txt  
streamlit run main.py
